A D3.js implementation of Conway's Game of Life. Click cells on the board to toggle them and between "alive" and "dead", and then hit Start to run the game. Alternatively, hit Random to pre-populate the board. Stop pauses the game, while Reset empties the board. Share or save the current board using the Generate URL or Share buttons. Or, start with any of these: Still Lifes, Blinkers and Beacons, Pulsars, Gosper Glider Gun.
